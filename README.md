## Usage

1. Open Profile Directory in [about:support](about:support)
2. [Create a `chrome` folder](https://www.userchrome.org/how-create-userchrome-css.html) in your Zen Browser's profile directory and configure it to enable userstyles.
3. Copy the `userChrome.css`, `userContent.css`, and `zen-logo.svg` files of your preferred theme from the [`themes/`](themes/) directory in this repository to your `chrome` folder.
4. Enable `toolkit.legacyUserProfileCustomizations.stylesheets` in [about:config](about:config).
